*Type Annotation
Type annotation is when you explicitly specify the type.
EXAMPLE
let age: number = 17;
let name: string = "Ronny";

function add(a: number, b: number): number {
  return a + b;
}

*Type inference
Type inference is when TypeScript automatically guesses the type based on the value.
let city = "Karachi"; // inferred as string
let score = 95;      // inferred as number
